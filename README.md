# OIBSIP Web Development and Designing Internship

## Task 1: Landing Page

### Objective

The objective of this task was to design and develop a visually appealing and responsive landing page using HTML and CSS. The landing page was created to provide a modern user experience while demonstrating frontend development and design skills.

### Project Title

AgroVision – AI-Powered Plant Disease Detection Landing Page

### Steps Performed

1. Designed the overall layout and structure of the landing page.
2. Created a responsive navigation bar.
3. Developed an attractive hero section with call-to-action buttons.
4. Added a features section highlighting key functionalities.
5. Implemented a "How It Works" section.
6. Created an About section describing the project.
7. Added statistics and achievements cards.
8. Designed a contact form for user interaction.
9. Implemented hover effects, animations, and responsive design using CSS.
10. Tested the website across different screen sizes.

### Tools & Technologies Used

* HTML5
* CSS3
* Font Awesome
* Google Fonts
* Visual Studio Code

### Outcome

Successfully developed a modern and responsive landing page for an AI-powered agriculture platform. The project demonstrates skills in webpage layout design, responsive web development, UI styling, and user experience enhancement.

* Responsive Design
* Modern Hero Section
* Navigation Bar
